## INSTALLATION ON LINUX
### Arch based
```bash
> sudo pacman -Syu
> sudo pacman -Sy vim
```
### Debian based
```bash
> sudo apt update && apt upgrade
> sudo apt install vim -y
```
### Rpm based
```bash 
> su 
> yum install vim
```
